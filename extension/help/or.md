# Arabic Reader - ବ୍ୟବହାରକାରୀ ଗାଇଡ୍

> ସଂସ୍କରଣ: v1.0.1

ଓଡ଼ିଆ ଅନୁବାଦ ପ୍ରସ୍ତୁତିରେ ଅଛି। ବର୍ତ୍ତମାନ ପାଇଁ ନିମ୍ନରେ ଇଂରାଜୀ ଗାଇଡ୍‌ର ସଂକ୍ଷିପ୍ତ ତଥ୍ୟ ଦିଆଯାଇଛି।

## ଦ୍ରୁତ ଆରମ୍ଭ

1. ବ୍ରାଉଜରରେ Arabic Reader Extension ଇନସ୍ଟଲ୍ କରନ୍ତୁ।
2. ଆରବିକ୍ ଟେକ୍ସଟ୍ ଥିବା ପେଜ୍ ଖୋଲନ୍ତୁ।
3. Popup ରେ **Enable Tashkeel** ଅନ୍ କରନ୍ତୁ।
4. ଆବଶ୍ୟକ ହେଲେ **Whole Page Tashkeel** ଅନ୍ କରନ୍ତୁ।

## ମୁଖ୍ୟ ବିଶେଷତା

- ଆରବିକ୍ ଶବ୍ଦ ପାଇଁ ସ୍ୱୟଂଚାଳିତ Tashkeel
- Hover dictionary (ଅର୍ଥ, root, I'rab forms)
- Text-to-speech ଏବଂ karaoke highlighting
- ଚୟନିତ ଟେକ୍ସଟ୍ ପାଇଁ translation
- Built-in PDF reader ଏବଂ PDF smart detection
