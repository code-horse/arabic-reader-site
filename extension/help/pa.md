# Arabic Reader - ਵਰਤੋਂਕਾਰ ਗਾਈਡ

> ਵਰਜਨ: v1.0.1

ਪੰਜਾਬੀ ਅਨੁਵਾਦ ਤਿਆਰ ਕੀਤਾ ਜਾ ਰਿਹਾ ਹੈ। ਇਸ ਵੇਲੇ ਹੇਠਾਂ ਅੰਗਰੇਜ਼ੀ ਗਾਈਡ ਦਾ ਸੰਖੇਪ ਦਿੱਤਾ ਗਿਆ ਹੈ।

## ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

1. ਬ੍ਰਾਊਜ਼ਰ ਵਿੱਚ Arabic Reader Extension ਇੰਸਟਾਲ ਕਰੋ।
2. ਅਰਬੀ ਲਿਖਤ ਵਾਲਾ ਵੈੱਬ ਪੇਜ ਖੋਲ੍ਹੋ।
3. Popup ਵਿੱਚ **Enable Tashkeel** ਚਾਲੂ ਕਰੋ।
4. ਲੋੜ ਅਨੁਸਾਰ **Whole Page Tashkeel** ਚਾਲੂ ਕਰੋ।

## ਮੁੱਖ ਫੀਚਰ

- ਅਰਬੀ ਸ਼ਬਦਾਂ ਲਈ ਆਟੋਮੈਟਿਕ Tashkeel
- Hover dictionary (ਅਰਥ, root, I'rab forms)
- Text-to-speech ਅਤੇ karaoke highlighting
- ਚੁਣੇ ਟੈਕਸਟ ਲਈ translation
- Built-in PDF reader ਅਤੇ PDF smart detection
