# Arabic Reader — Руководство пользователя

> Версия: v1.0.0

## Введение

Arabic Reader — расширение для браузера для изучающих арабский язык. Оно добавляет огласовку Tashkeel (حركات) к арабским словам на веб-страницах и в PDF: автоматическое огласование, словарь при наведении (корень и грамматические формы), озвучивание и перевод — для удобного изучения произношения и грамматики.

---

## Основные возможности

- **Авто Tashkeel** — Огласовочные знаки (fathah, kasrah, dammah и др.) для слов на любых сайтах
- **Режим всей страницы** — Одним щелчком огласовка всех арабских слов на странице
- **Словарь при наведении** — При наведении: огласовка, значения, корень и формы I'rab; режимы «Словарь», «Подсказка» или «Выкл.»
- **Разбор I'rab** — Грамматические окончания (им., вин., род. п.) для подходящих слов
- **Синтез речи** — Арабская TTS с подсветкой слов (голос ar-SA)
- **Озвучивание выделения** — После выделения текста компактная панель: чтение и перевод
- **Перевод** — Выделите текст и нажмите «Перевод» для мгновенного перевода через Bing или Google (108 языков)
- **Чтение PDF** — Встроенный RTL-просмотрщик PDF с поддержкой Tashkeel
- **Умное обнаружение** — Автоопределение арабского текста и перенаправление PDF
- **Выделение корня** — Для понимания морфологии
- **Горячие клавиши**
- **Многоязычный интерфейс** — Арабский, английский, китайский

## Как пользоваться

### Шаг 1: Установка

Установите **Arabic Reader** из [Chrome Web Store](https://chromewebstore.google.com/) или загрузите локально в режиме разработчика.

### Шаг 2: Откройте страницу

Откройте сайт с арабским текстом.

### Шаг 3: Включите Tashkeel

Нажмите на иконку расширения. Включите «Включить Tashkeel», затем «Tashkeel для всей страницы».

### Шаг 4: Просмотр подсказок

Наведите курсор на слово, чтобы увидеть всплывающую подсказку с огласовкой и значениями. Нажмите на значок динамика.

### Шаг 5: Озвучивание и перевод выделения

Выделите арабский текст. Появится панель:
- **Динамик** — Читает выделение с подсветкой по словам
- **Перевод** — Показывает встроенную подсказку перевода под панелью

> **Совет:** Через иконку расширения откройте настройки (режим наведения, скорость речи, движок перевода и т.д.).

## Режим Tashkeel для всей страницы

При включении каждое арабское слово получает огласовку. Расширение находит блоки арабского текста и применяет Tashkeel с помощью встроенного словаря и NLP.

---

## Словарь при наведении

Встроенный арабский словарь. Режимы в настройках:

| Режим | Поведение |
|-------|-----------|
| **Словарь** | Наведение: Tashkeel + значения + корень + I'rab + кнопка озвучивания |
| **Подсказка** | Наведение: Tashkeel + кнопка (без значений) |
| **Выкл.** | Эффекта нет |

В режиме **Словарь** подсказка показывает огласованное слово, кнопку произношения, значения, трёхкоренной корень и I'rab при включении.

## Просмотрщик PDF

Встроенный просмотрщик с полной поддержкой RTL для арабских PDF с Tashkeel.

### Открытие PDF

**1 — Из всплывающего окна:** иконка → «Открыть просмотрщик PDF», перетаскивание файла или «Выбрать файл», можно вставить URL PDF.

**2 — Контекстное меню:** ПКМ по ссылке `.pdf` → «Открыть PDF в Arabic Reader».

**3 — Автообнаружение:** при «Умном обнаружении PDF» адреса `.pdf` перенаправляются во встроенный просмотрщик.

### Возможности PDF

- **Аннотации Tashkeel** — На тексте PDF
- **Словарь по клику**
- **Панель выделения** — Чтение, перевод, копирование
- **Боковая панель** — Оглавление и миниатюры
- **Поиск**
- **Темы** — Тёмная, светлая, сепия
- **Масштаб** — Авто, страница, по ширине
- **RTL-вёрстка**

## Синтез речи

Используется встроенный в Chrome арабский голос TTS (ar-SA).

**Слово:** наведите и нажмите динамик во всплывающей подсказке.

**Выделение:** плавающая панель с динамиком и подсветкой слов.

**Контекстное меню:** «Прочитать выделенное».

**Сочетание:** выделите текст и нажмите `Alt+Shift+S` (на Mac: `Ctrl+Shift+S`).

---

## Перевод

Выделите текст для мгновенного перевода.

**1:** Панель выделения → Перевод.

**2:** Контекстное меню → «Перевести выделенное».

**3:** `Alt+Shift+T` (Mac: `Ctrl+Shift+T`).

**Движки:** **Bing Переводчик** (по умолчанию), **Google Переводчик**. Оба поддерживают **108 языков**.

## Частые вопросы

**Почему не работает на некоторых страницах?**  
Из соображений безопасности расширения не выполняются на `chrome://`, в настройках браузера и в Chrome Web Store.

**Нет звука TTS?**  
Проверьте громкость и установленные арабские голоса. Поддержка зависит от браузера и ОС.

**Арабский не определяется.**  
Текст в виде изображения или с нестандартной кодировкой может не распознаваться.

**Перевод не работает?**  
Нужно подключение к интернету. При сбое Bing переключитесь на Google в настройках.

**Как открыть PDF в Arabic Reader?**  
«Открыть просмотрщик PDF» во всплывающем окне, контекстное меню по ссылке PDF или включите «Умное обнаружение PDF».
