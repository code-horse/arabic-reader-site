# Arabic Reader - 使用手冊

> 版本：v1.0.0

## 簡介

Arabic Reader 是為阿拉伯語學習者設計的瀏覽器擴充功能。它會為網頁與 PDF 中的阿拉伯語單詞加上 Tashkeel（母音符號／حركات），提供自動標音、含詞根分析與文法形式的懸停字典、文字轉語音與翻譯功能，協助您更輕鬆地學習阿拉伯語發音與文法。

---

## 主要功能

- **自動 Tashkeel** — 在任意網頁上為阿拉伯語單詞加上母音標記（fathah、kasrah、dammah、sukun、shadda、tanween）
- **整頁模式** — 一鍵為頁面上所有阿拉伯語單詞加上標註
- **懸停字典** — 將游標移至單詞上可查看標音、釋義、詞根分析與 I'rab 形式；可選擇字典模式、工具提示模式或關閉
- **I'rab 分析** — 顯示適用單詞的文法格變化（主格、賓格、屬格）
- **文字轉語音** — 阿拉伯語 TTS，逐字卡拉 OK 高亮（ar-SA 語音）
- **選取朗讀** — 選取任意文字後會出現精簡工具列，含朗讀與翻譯按鈕
- **翻譯** — 選取任意文字，按翻譯即可透過 Bing 或 Google 翻譯取得即時翻譯（108 種語言）
- **PDF 閱讀器** — 內建 RTL PDF 檢視器，支援 Tashkeel 標註
- **智慧偵測** — 自動偵測阿拉伯語文字與 PDF 智慧重新導向
- **詞根擷取** — 辨識詞根，協助理解阿拉伯語詞形變化
- **鍵盤快捷鍵** — 快速使用核心功能
- **多語介面** — 阿拉伯語、英語與中文介面

## 使用方式

### 第一步：安裝擴充功能

從 [Chrome 線上應用程式商店](https://chromewebstore.google.com/) 安裝 **Arabic Reader**，或以開發人員模式本機載入。

### 第二步：開啟網頁

前往任何包含阿拉伯語內容的網頁。

### 第三步：啟用 Tashkeel

點選瀏覽器工具列中的擴充功能圖示。開啟「啟用 Tashkeel」，再開啟「整頁 Tashkeel」為頁面上所有阿拉伯語單詞加上標註。

### 第四步：查看標註

將游標懸停在單詞上以查看含標音、釋義與發音的 Tashkeel 提示。按一下喇叭圖示可聽取單詞。

### 第五步：朗讀與翻譯選取文字

用滑鼠選取任意阿拉伯語文字。選取範圍附近會顯示精簡工具列，有兩個按鈕：
- **喇叭** — 以卡拉 OK 風格逐字高亮朗讀選取內容
- **翻譯** — 在工具列下方顯示內嵌翻譯氣泡

> **提示：** 點選瀏覽器工具列中的擴充功能圖示可開啟設定面板，調整懸停模式、語速、翻譯引擎等。

## 整頁 Tashkeel 模式

啟用整頁 Tashkeel 時，頁面上每個阿拉伯語單詞都會加上母音標記。擴充功能使用智慧偵測找出阿拉伯語文字區塊，並以內建字典與 NLP 演算法套用 Tashkeel。

---

## 懸停字典

擴充功能內建阿拉伯語字典。您可在設定中選擇多種懸停模式：

| 模式 | 行為 |
|------|------|
| **字典** | 懸停顯示 Tashkeel + 釋義 + 詞根 + I'rab + 發音按鈕 |
| **工具提示** | 懸停顯示 Tashkeel + 發音按鈕（無釋義） |
| **關閉** | 無懸停效果 |

在 **字典模式**下，提示包含：
- 完整標音的單詞（Tashkeel）
- 發音按鈕（按一下可聆聽）
- 詞義
- 詞根分析（三母根）
- 啟用時顯示 I'rab（文法格）形式

## PDF 閱讀器

Arabic Reader 內建完整支援 RTL 的 PDF 閱讀器，可閱讀含 Tashkeel 標註的阿拉伯語 PDF。

### 開啟 PDF

**方式一：從彈出視窗**  
點選擴充功能圖示，再按「開啟 PDF 閱讀器」。拖放 PDF 檔案或按「選擇檔案」開啟本機 PDF。亦可貼上 PDF URL。

**方式二：情境選單**  
在網頁上對任一 `.pdf` 連結按右鍵，選擇「以 Arabic Reader 開啟 PDF」。

**方式三：自動偵測**  
在設定中啟用「PDF 智慧偵測」時，擴充功能會自動將 `.pdf` URL 重新導向內建閱讀器。

### PDF 閱讀器功能

- **Tashkeel 標註** — 所有標註功能適用於 PDF 文字
- **點擊字典** — 點選任意單詞查看釋義
- **選取工具列** — 選取文字以朗讀、翻譯或複製
- **側邊欄** — 目錄大綱與頁面縮圖
- **搜尋** — 在 PDF 中搜尋文字
- **主題** — 深色、淺色與仿古紙閱讀主題
- **縮放** — 多種縮放層級，含自動、整頁與頁寬
- **RTL 版面** — 完整由右至左排版

## 文字轉語音

擴充功能使用 Chrome 內建的阿拉伯語 TTS（ar-SA）語音發音。

**單一字詞：** 將游標懸停在單詞上，在提示中按喇叭按鈕。

**選取朗讀：** 選取任意阿拉伯語文字。會出現浮動工具列與喇叭按鈕 — 按一下即可朗讀，並以卡拉 OK 風格逐字高亮。

**右鍵選單：** 選取文字，按右鍵並選「朗讀選取範圍」。

**鍵盤快捷鍵：** 選取文字後按 `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）。

---

## 翻譯

選取頁面上任意文字即可使用翻譯功能取得即時翻譯。

**方式一：選取工具列**  
選取文字後，按工具列中的翻譯按鈕。

**方式二：右鍵選單**  
選取文字，按右鍵並選「翻譯選取範圍」。

**方式三：鍵盤快捷鍵**  
選取文字後按 `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）。

**翻譯引擎：**
- **Bing Translate**（預設）— 由 Microsoft Translator 提供
- **Google Translate** — 由 Google 提供

兩種引擎皆支援 **108 種目標語言**。

## 鍵盤快捷鍵

| 快捷鍵 | Mac 快捷鍵 | 動作 |
|--------|-----------|------|
| `Alt+Shift+A` | `Ctrl+Shift+A` | 開關 Tashkeel 標註 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 朗讀選取文字 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 翻譯選取文字 |

> **提示：** 可在 Chrome 的 `chrome://extensions/shortcuts` 自訂這些快捷鍵。

## 設定說明

| 設定 | 說明 |
|------|------|
| **啟用 Tashkeel** | 主開關，啟用或停用 Tashkeel 標註功能 |
| **整頁 Tashkeel** | 啟用後為頁面上所有阿拉伯語單詞加上母音標記 |
| **懸停模式** | 選擇懸停行為：字典（含釋義與音訊）、工具提示（Tashkeel + 音訊）或關閉 |
| **顯示 I'rab 形式** | 顯示適用單詞的文法格結尾 |
| **語速** | 調整句子朗讀速度 |
| **翻譯引擎** | 選擇 Bing Translate 或 Google Translate |
| **目標語言** | 設定翻譯目標語言 |
| **PDF 智慧偵測** | 啟用後自動將 PDF URL 重新導向至內建閱讀器 |

## 常見問題

**問：為什麼有些頁面無法使用？**  
答：基於安全性，瀏覽器擴充功能無法在 `chrome://`、瀏覽器設定或 Chrome 線上應用程式商店等特殊頁面執行。

**問：文字轉語音沒有聲音？**  
答：請檢查系統音量，並確認已安裝阿拉伯語語音套件。語音支援因瀏覽器與作業系統而異。

**問：擴充功能偵測不到阿拉伯語文字。**  
答：擴充功能會智慧偵測阿拉伯語文字區塊。若文字以圖片呈現或使用非標準編碼，可能無法偵測。

**問：翻譯無法運作？**  
答：翻譯需要網路連線。若 Bing Translate 失敗，請在設定中改用 Google Translate。

**問：如何以 Arabic Reader 開啟 PDF？**  
答：在彈出視窗中按「開啟 PDF 閱讀器」、對 PDF 連結按右鍵選「以 Arabic Reader 開啟」，或在設定中啟用「PDF 智慧偵測」。
