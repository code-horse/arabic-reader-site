# Arabic Reader - ಬಳಕೆದಾರ ಮಾರ್ಗದರ್ಶಿ

> ಆವೃತ್ತಿ: v1.0.1

ಕನ್ನಡ ಅನುವಾದ ಸಿದ್ಧವಾಗುತ್ತಿದೆ. ಈಗಿಗೆ ಕೆಳಗೆ ಇಂಗ್ಲಿಷ್ ಮಾರ್ಗದರ್ಶಿಯ ಸಂಕ್ಷಿಪ್ತ ಆವೃತ್ತಿ ನೀಡಲಾಗಿದೆ.

## ತ್ವರಿತ ಆರಂಭ

1. ಬ್ರೌಸರ್‌ನಲ್ಲಿ Arabic Reader Extension ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿ.
2. ಅರಬಿಕ್ ಪಠ್ಯವಿರುವ ವೆಬ್ ಪುಟವನ್ನು ತೆರೆಯಿರಿ.
3. Popup ನಲ್ಲಿ **Enable Tashkeel** ಅನ್ನು ಆನ್ ಮಾಡಿ.
4. ಅಗತ್ಯವಿದ್ದರೆ **Whole Page Tashkeel** ಅನ್ನು ಆನ್ ಮಾಡಿ.

## ಮುಖ್ಯ ವೈಶಿಷ್ಟ್ಯಗಳು

- ಅರಬಿಕ್ ಪದಗಳಿಗೆ ಸ್ವಯಂ Tashkeel
- Hover dictionary (ಅರ್ಥ, root, I'rab forms)
- Text-to-speech ಮತ್ತು karaoke highlighting
- ಆಯ್ಕೆ ಮಾಡಿದ ಪಠ್ಯಕ್ಕೆ translation
- Built-in PDF reader ಮತ್ತು PDF smart detection
