# Arabic Reader - વપરાશકર્તા માર્ગદર્શિકા

> સંસ્કરણ: v1.0.1

ગુજરાતી અનુવાદ પ્રક્રિયામાં છે. હાલ માટે નીચે અંગ્રેજી માર્ગદર્શિકાની સંક્ષિપ્ત માહિતી આપવામાં આવે છે.

## ઝડપી શરૂઆત

1. બ્રાઉઝરમાં Arabic Reader Extension ઇન્સ્ટોલ કરો.
2. Arabic લખાણ ધરાવતી વેબસાઇટ ખોલો.
3. Popup માં **Enable Tashkeel** ચાલુ કરો.
4. જરૂરી હોય તો **Whole Page Tashkeel** ચાલુ કરો.

## મુખ્ય સુવિધાઓ

- Arabic શબ્દો માટે સ્વચાલિત Tashkeel
- Hover dictionary (અર્થ, root, I'rab forms)
- Text-to-speech અને karaoke highlighting
- પસંદ કરેલા લખાણ માટે translation
- Built-in PDF reader અને PDF smart detection
