# Arabic Reader - Användarhandbok

> Version: v1.0.0

## Introduktion

Arabic Reader är ett webbläsartillägg designat för arabiska elever. Den lägger till Tashkeel (vokalmärken / حركات) till arabiska ord på webbsidor och PDF-filer, vilket ger automatisk vokalisering, en hovringsordbok med rotanalys och grammatiska former, text-till-tal och översättningsfunktioner – vilket hjälper dig att lättare lära dig arabiskt uttal och grammatik.

---

## Huvudfunktioner

- **Automatisk Tashkeel** — Lägger till vokaltecken (fathah, kasrah, dammah, sukun, shadda, tanween) till arabiska ord på vilken webbsida som helst
- **Helsidesläge** — Ett klick Tashkeel för alla arabiska ord på sidan
- **Hover Dictionary** — Håll muspekaren över ord för att se vokalisering, definitioner, rotanalys och I'rab-former; välj mellan ordboksläge, verktygstipsläge eller av
- **I'rab Analys** — Visa grammatiska kasusändelser (nominativ, ackusativ, genitiv) för tillämpliga ord
- **Text-till-tal** — Arabisk TTS med ord-för-ord karaokemarkering (ar-SA-röst)
- **Markeringstal** — Välj valfri text, ett kompakt verktygsfält visas med knappar för uppläsning och översättning
- **Översättning** — Välj valfri text, klicka på översätt för att få omedelbar översättning via Bing eller Google Translate (108 språk)
- **PDF-läsare** — Inbyggd RTL PDF-visare med Tashkeel annoteringsstöd
- **Smart identifiering** — Automatisk arabisk textidentifiering och smart omdirigering av PDF
- **Root Extraction** — Identifiera ordrötter för bättre förståelse av arabisk morfologi
- **Kortkommandon** — Snabb åtkomst till centrala funktioner
- **Flerspråkigt gränssnitt** — arabiska, engelska och kinesiska användargränssnitt

## Hur man använder

### Steg 1: Installera tillägget

Installera **Arabic Reader** från [Chrome Web Store](https://chromewebstore.google.com/), eller ladda det lokalt i utvecklarläge.

### Steg 2: Öppna valfri webbsida

Besök valfri webbsida som innehåller arabiskt innehåll.

### Steg 3: Aktivera Tashkeel

Klicka på tilläggsikonen i webbläsarens verktygsfält. Aktivera "Aktivera Tashkeel" och sedan "Hel sidan Tashkeel" för att kommentera alla arabiska ord på sidan.

### Steg 4: Visa kommentarer

Håll muspekaren över ord för att se Tashkeel verktygstips med vokalisering, definitioner och uttal. Klicka på högtalarikonen för att höra ordet.

### Steg 5: Tala och översätt vald text

Välj valfri arabisk text med musen. Ett kompakt verktygsfält visas nära markeringen med två knappar:
- **Högtalare** — Läser den valda texten högt med karaoke-stil ord-för-ord-markering
- **Översätt** — Visar en inbäddad översättningsbubbla under verktygsfältet

> **Tips:** Klicka på tilläggsikonen i webbläsarens verktygsfält för att öppna inställningspanelen och justera hovringsläge, talhastighet, översättningsmotor med mera.

## Hela sidan Tashkeel Läge

När läget för helsida Tashkeel är aktiverat får varje arabiskt ord på sidan vokaltecken. Tillägget använder smart detektering för att identifiera arabiska textblock och tillämpar Tashkeel med hjälp av dess medföljande ordbok och NLP-algoritmer.

---

## Hover Dictionary

Tillägget innehåller en inbyggd arabisk ordbok. Du kan välja mellan flera svävningslägen i inställningarna:

| Läge | Beteende |
|------|--------|
| **Ordbok** | Hover visar Tashkeel + definitioner + root + I'rab + uttalsknapp |
| **Verktygstips** | Hover visar Tashkeel + uttalsknapp (inga definitioner) |
| **Av** | Ingen svävningseffekt |

I **Ordboksläge** visar verktygstipset:
- Ordet med full vokalisering (Tashkeel)
- En uttalsknapp (klicka för att höra)
- Orddefinitioner
- Rotanalys (rot på tre bokstäver)
- I'rab (grammatiska skiftlägen) bildas när det är aktiverat

## PDF-läsare

Arabic Reader innehåller en inbyggd PDF-läsare med fullt RTL-stöd som låter dig läsa arabiska PDF-dokument med Tashkeel-anteckningar.

### Öppna en PDF

**Metod 1: Från popupen**  
Klicka på tilläggsikonen och klicka sedan på "Öppna PDF Reader". Dra och släpp en PDF-fil eller klicka på "Välj fil" för att öppna en lokal PDF. Du kan också klistra in en PDF-URL.

**Metod 2: Snabbmeny**  
Högerklicka på valfri `.pdf`-länk på en webbsida och välj "Öppna PDF med Arabic Reader".

**Metod 3: Automatisk upptäckt**  
När "PDF Smart Detection" är aktiverat i inställningarna, omdirigerar tillägget automatiskt `.pdf`-URL:er till den inbyggda läsaren.

### PDF-läsarfunktioner

- **Tashkeel Anteckningar** — Alla anteckningsfunktioner fungerar på PDF-text
- **Klicka på Ordbok** — Klicka på ett ord för att se dess definition
- **Markeringsverktygsfält** — Välj text att tala, översätta eller kopiera
- **Sidofält** — Översikt över innehållsförteckningen och sidminiatyrer
- **Sök** — Sök efter text i PDF:en
- **Teman** — Mörka, Ljusa och Sepia lästeman
- **Zoom** — Flera zoomnivåer inklusive Auto, Sidanpassning och Sidbredd
- **RTL-layout** — Fullständigt stöd från höger till vänster

## Text-till-tal

Tillägget använder Chromes inbyggda arabiska TTS-röst (ar-SA) för uttal.

**Enstaka ord:** Håll muspekaren över ett ord och klicka på högtalarknappen i verktygstipset.

**Val tal:** Välj valfri arabisk text. Ett flytande verktygsfält visas med en högtalarknapp — klicka på den för att läsa upp med karaoke-stil ord-för-ord-markering.

**Högerklicksmeny:** Välj text, högerklicka och välj "Säg upp markering".

**Kortkommando:** Välj text och tryck på `Alt+Shift+S` (Mac: `Ctrl+Shift+S`).

---

## Översättning

Välj valfri text på sidan och använd översättningsfunktionen för att få omedelbara översättningar.

**Metod 1: Markeringsverktygsfält**  
Markera text och klicka sedan på översättknappen i verktygsfältet.

**Metod 2: Högerklicka på menyn**  
Markera text, högerklicka och välj "Översätt urval".

**Metod 3: Kortkommando**  
Välj text och tryck på `Alt+Shift+T` (Mac: `Ctrl+Shift+T`).

**Översättningsmotorer:**
- **Bing Translate** (standard) — Drivs av Microsoft Translator
- **Google Translate** — Drivs av Google

Båda motorerna stöder **108 målspråk**.

## Vanliga frågor

**F: Varför fungerar det inte på vissa sidor?**  
S: Av säkerhetsskäl kan webbläsartillägg inte köras på speciella sidor som `chrome://`, webbläsarinställningar eller Chrome Web Store.

**F: Inget ljud från text-till-tal?**  
S: Kontrollera dina systemvolyminställningar och se till att arabiska röstpaket är installerade. Talstödet varierar mellan webbläsare och operativsystem.

**F: Tillägget upptäcker inte arabisk text.**  
S: Tillägget använder smart identifiering för att hitta arabiska textblock. Om texten renderas som bilder eller använder icke-standardkodning kanske den inte upptäcks.

**F: Fungerar inte översättningen?**  
S: Översättning kräver en internetanslutning. Om Bing Translate misslyckas, försök växla till Google Translate i inställningarna.

**F: Hur öppnar jag en PDF med Arabic Reader?**  
S: Klicka på "Öppna PDF Reader" i popup-fönstret, högerklicka på en PDF-länk och välj "Öppna PDF med Arabic Reader", eller aktivera "PDF Smart Detection" i inställningarna.
