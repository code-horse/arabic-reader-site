# Arabic Reader - वापरकर्ता मार्गदर्शक

> आवृत्ती: v1.0.1

मराठी अनुवाद प्रक्रियेत आहे. सध्या खाली इंग्रजी मार्गदर्शकाचा संक्षेप दिला आहे.

## झटपट सुरुवात

1. ब्राउझरमध्ये Arabic Reader Extension इन्स्टॉल करा.
2. अरबी मजकूर असलेले पान उघडा.
3. Popup मध्ये **Enable Tashkeel** चालू करा.
4. गरजेनुसार **Whole Page Tashkeel** चालू करा.

## मुख्य वैशिष्ट्ये

- अरबी शब्दांसाठी स्वयंचलित Tashkeel
- Hover dictionary (अर्थ, root, I'rab forms)
- Text-to-speech आणि karaoke highlighting
- निवडलेल्या मजकुरासाठी translation
- Built-in PDF reader आणि PDF smart detection
