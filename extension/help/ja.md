# Arabic Reader - ユーザーガイド

> バージョン: v1.0.0

## はじめに

Arabic Reader は、アラビア語学習者向けのブラウザ拡張機能です。Webページや PDF のアラビア語単語に Tashkeel（母音符／حركات）を付与し、自動発音記号付け、ホバー辞典（語根解析と文法形）、音声読み上げ、翻訳により、発音と文法の習得を支援します。

---

## 主な機能

- **自動 Tashkeel** — 任意の Web ページ上のアラビア語に母音記号（fathah、kasrah、dammah、sukun、shadda、tanween）を付与
- **ページ全体モード** — ワンクリックでページ内のすべてのアラビア語に Tashkeel
- **ホバー辞典** — 単語にマウスを合わせて発音記号・定義・語根解析・ I'rab 形を表示。辞典／ツールヒント／オフから選択可能
- **I'rab 分析** — 該当する語に対して文法格（主格・対格・属格など）を表示
- **テキスト読み上げ** — アラビア語 TTS、単語ごとのカラオケ風ハイライト（ar-SA 音声）
- **選択範囲読み上げ** — テキスト選択後にコンパクトなツールバーから読み上げ／翻訳
- **翻訳** — 任意のテキストを選択し、Bing または Google 翻訳で即時翻訳（108言語対象）
- **PDF ビューワー** — Tashkeel 対応の RTL 統合 PDF ビューワー
- **スマート検出** — アラビア語検出と PDF の自動転送
- **語根抽出** — 形態論の理解を支援
- **キーボードショートカット**
- **多言語 UI** — アラビア語、英語、中国語

## 使い方

### ステップ 1：拡張機能をインストール

[Chrome ウェブストア](https://chromewebstore.google.com/) から **Arabic Reader** をインストールするか、開発者モードでローカル読み込みしてください。

### ステップ 2：Webページを開く

アラビア語を含むページを開きます。

### ステップ 3：Tashkeel を有効化

ブラウザの拡張機能アイコンをクリックします。「Tashkeel を有効にする」をオンにし、「ページ全体 Tashkeel」でページ上のすべてのアラビア語に注記を付けます。

### ステップ 4：注記を確認

単語にマウスを合わせると、発音記号・定義・発音情報を含む Tashkeel ツールチップが表示されます。スピーカーアイコンで単語を聞けます。

### ステップ 5：選択範囲の読み上げと翻訳

アラビア語テキストを選択すると、近くにコンパクトなツールバーが表示されます。
- **スピーカー** — 選択範囲を単語ごとにハイライトしながら読み上げ
- **翻訳** — ツールバーの下にインライン翻訳バブルを表示

> **ヒント：** 拡張機能アイコンから設定パネルを開き、ホバーモード、読み上げ速度、翻訳エンジンなどを調整できます。

## ページ全体 Tashkeel モード

有効にすると、ページ上の各アラビア語に母音が付きます。拡張機能はアラビア語ブロックを検出し、内蔵辞書と NLP で Tashkeel を適用します。

---

## ホバー辞典

内蔵アラビア語辞典。設定でホバーモードを選択します。

| モード | 動作 |
|--------|------|
| **辞典** | ホバーで Tashkeel + 定義 + 語根 + I'rab + 発音ボタン |
| **ツールヒント** | ホバーで Tashkeel + 発音ボタン（定義なし） |
| **オフ** | ホバー効果なし |

**辞典**モードでは、発音記号付き単語、発音ボタン、定義、三子音語根、有効時は I'rab が表示されます。

## PDF ビューワー

RTL 完全対応の統合 PDF ビューワーで、Tashkeel 付きアラビア語 PDF を閲覧できます。

### PDF を開く

**方法1：ポップアップ** — 拡張機能アイコン →「PDF ビューワーを開く」。ファイルをドラッグ＆ドロップまたは「ファイルを選択」。PDF の URL を貼り付け可能。

**方法2：コンテキストメニュー** — `.pdf` リンクを右クリックし、「Arabic Reader で PDF を開く」を選択。

**方法3：自動検出** — 設定で「PDF スマート検出」をオンにすると、`.pdf` の URL が統合ビューワーに転送されます。

### PDF の機能

- **Tashkeel 注釈** — PDF テキストにも適用
- **クリック辞典** — 単語をクリックして定義
- **選択ツールバー** — 読み上げ・翻訳・コピー
- **サイドバー** — 目次とページサムネイル
- **検索**
- **テーマ** — ダーク、ライト、セピア
- **ズーム** — 自動、ページに合わせる、ページ幅など
- **RTL レイアウト**

## テキスト読み上げ

Chrome 組み込みのアラビア語 TTS（ar-SA）を使用します。

**単語：** ホバーしてツールチップのスピーカーをクリック。

**選択：** テキストを選択するとフローティングツールバーから読み上げ（カラオケ風ハイライト）。

**右クリックメニュー：**「選択範囲を読み上げ」。

**ショートカット：** 選択後 `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）。

---

## 翻訳

ページ上のテキストを選択して即時翻訳できます。

**1：** 選択ツールバーの翻訳ボタン。

**2：** 右クリック →「選択範囲を翻訳」。

**3：** `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）。

**エンジン：** **Bing 翻訳**（既定）、**Google 翻訳**。どちらも **108 の対象言語** に対応。

| ショートカット | Mac | 操作 |
|----------------|-----|------|

## 設定

| 設定 | 説明 |
|------|------|
| **Tashkeel を有効にする** | Tashkeel 注釈のマスタースイッチ |
| **ページ全体 Tashkeel** | ページ上のすべてのアラビア語に母音記号を付与 |
| **ホバーモード** | 辞典（定義+音声）、ツールヒント（Tashkeel+音声）、オフ |
| **I'rab 形を表示** | 該当語の文法辞尾を表示 |
| **読み上げ速度** | 文章読み上げの速さ |
| **翻訳エンジン** | Bing または Google |
| **対象言語** | 翻訳の目標言語 |
| **PDF スマート検出** | PDF の URL を統合ビューワーへ自動転送 |

## FAQ

**一部のページで動作しないのはなぜですか。**  
セキュリティ上、`chrome://`、ブラウザ設定、Chrome ウェブストアなどでは拡張機能は動作しません。

**音声読み上げが聞こえません。**  
システム音量とアラビア語音声パックを確認してください。ブラウザと OS によって対応が異なります。

**アラビア語が検出されません。**  
画像として描画されているテキストや特殊なエンコードでは検出できないことがあります。

**翻訳が動きません。**  
インターネット接続が必要です。Bing が失敗する場合は設定で Google に切り替えてください。

**Arabic Reader で PDF を開くには。**  
ポップアップの「PDF ビューワーを開く」、PDF リンクの右クリック、または「PDF スマート検出」を有効にしてください。
