# Arabic Reader - 사용자 가이드

> 버전: v1.0.0

## 소개

Arabic Reader는 아랍어 학습자를 위한 브라우저 확장 프로그램입니다. 웹 페이지와 PDF의 아랍어 단어에 Tashkeel(모음 부호 / حركات)을 더하고, 자동 발음 부호, 마우스 올림 사전(어근 분석·문법형), 텍스트 음성 변환(TTS), 번역 기능으로 발음과 문법 학습을 돕습니다.

---

## 주요 기능

- **자동 Tashkeel** — 모든 웹 페이지의 아랍어 단어에 모음 부호(fathah, kasrah, dammah, sukun, shadda, tanween) 추가
- **전체 페이지 모드** — 한 번의 클릭으로 페이지의 모든 아랍어에 Tashkeel 적용
- **호버 사전** — 단어 위에 마우스를 올리면 발음 기호·뜻·어근 분석·I'rab 형태 표시. 사전/툴팁/끔 선택 가능
- **I'rab 분석** — 해당 단어의 문법격 어미 표시
- **텍스트 음성 변환** — 아랍어 TTS, 단어별 카라오케 하이라이트(ar-SA 음성)
- **선택 영역 읽기** — 텍스트 선택 후 작은 도구 모음에서 읽기·번역
- **번역** — 텍스트를 선택하고 번역을 눌러 Bing 또는 Google 번역으로 즉시 번역(108개 목표 언어)
- **PDF 뷰어** — Tashkeel 지원 내장 RTL PDF 뷰어
- **스마트 감지** — 아랍어 자동 감지 및 PDF 자동 리디렉션
- **어근 추출** — 형태론 이해 지원
- **키보드 단축키**
- **다국어 UI** — 아랍어, 영어, 중국어

---

## 사용 방법

### 1단계: 확장 프로그램 설치

[Chrome 웹스토어](https://chromewebstore.google.com/)에서 **Arabic Reader**를 설치하거나 개발자 모드로 로컬에서 불러오세요.

### 2단계: 웹 페이지 열기

아랍어가 포함된 페이지를 방문합니다.

### 3단계: Tashkeel 사용

브라우저 도구 모음의 확장 프로그램 아이콘을 클릭합니다.「Tashkeel 사용」을 켜고「전체 페이지 Tashkeel」으로 페이지의 모든 아랍어에 표기합니다.

### 4단계: 주석 확인

단어에 마우스를 올리면 발음 기호·뜻·발음 정보가 있는 Tashkeel 툴팁이 표시됩니다. 스피커 아이콘으로 단어를 들을 수 있습니다.

### 5단계: 선택 영역 읽기 및 번역

아랍어 텍스트를 선택하면 근처에 작은 도구 모음이 나타납니다.
- **스피커** — 선택 내용을 단어별로 하이라이트하며 읽기
- **번역** — 도구 모음 아래에 인라인 번역 말풍선 표시

> **팁:** 확장 프로그램 아이콘을 클릭해 호버 모드, 읽기 속도, 번역 엔진 등을 조정할 수 있습니다.

## 전체 페이지 Tashkeel 모드

이 모드를 켜면 페이지의 모든 아랍어 단어에 모음 부호가 붙습니다. 확장 프로그램이 아랍어 블록을 찾아 내장 사전과 NLP로 Tashkeel을 적용합니다.

---

## 호버 사전

내장 아랍어 사전. 설정에서 호버 모드를 선택합니다.

| 모드 | 동작 |
|------|------|
| **사전** | 호버 시 Tashkeel + 정의 + 어근 + I'rab + 발음 버튼 |
| **툴팁** | 호버 시 Tashkeel + 발음 버튼(정의 없음) |
| **끔** | 호버 효과 없음 |

**사전** 모드에서는 발음 기호가 붙은 단어, 발음 버튼, 정의, 삼자 어근, 설정 시 I'rab이 표시됩니다.

## PDF 뷰어

Tashkeel 주석이 가능한 내장 RTL PDF 뷰어로 아랍어 PDF를 읽을 수 있습니다.

### PDF 열기

**방법 1: 팝업** — 확장 프로그램 아이콘 →「PDF 뷰어 열기」. 파일을 끌어다 놓거나「파일 선택」. PDF URL 붙여넣기 가능.

**방법 2: 상황에 맞는 메뉴** — `.pdf` 링크를 오른쪽 클릭 →「Arabic Reader로 PDF 열기」.

**방법 3: 자동 감지** — 설정에서「PDF 스마트 감지」를 켜면 `.pdf` URL이 내장 뷰어로 이동합니다.

### PDF 기능

- **Tashkeel 주석** — PDF 텍스트에도 동일
- **클릭 사전** — 단어 클릭으로 정의
- **선택 도구 모음** — 읽기·번역·복사
- **사이드바** — 목차 및 페이지 썸네일
- **검색**
- **테마** — 어둡게, 밝게, 세피아
- **확대/축소** — 자동, 페이지 맞춤, 페이지 너비 등
- **RTL 레이아웃**

## 텍스트 음성 변환

Chrome에 포함된 아랍어 TTS(ar-SA) 음성을 사용합니다.

**단일 단어:** 호버 후 툴팁의 스피커 클릭.

**선택:** 텍스트 선택 후 떠 있는 도구 모음에서 읽기(카라오케 스타일 하이라이트).

**오른쪽 클릭 메뉴:**「선택 영역 읽기」.

**단축키:** 선택 후 `Alt+Shift+S`(Mac: `Ctrl+Shift+S`).

---

## 번역

페이지에서 텍스트를 선택해 즉시 번역합니다.

**1:** 선택 도구 모음의 번역 버튼.

**2:** 오른쪽 클릭 →「선택 영역 번역」.

**3:** `Alt+Shift+T`(Mac: `Ctrl+Shift+T`).

**엔진:** **Bing 번역**(기본), **Google 번역**. 둘 다 **108개 목표 언어** 지원.

---

## 키보드 단축키

| 단축키 | Mac 단축키 | 동작 |
|--------|------------|------|
| `Alt+Shift+A` | `Ctrl+Shift+A` | Tashkeel 켜기/끄기 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 선택 영역 읽기 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 선택 영역 번역 |

> **팁:** `chrome://extensions/shortcuts`에서 사용자 지정할 수 있습니다.

## 자주 묻는 질문

**일부 페이지에서 동작하지 않는 이유는?**  
보안상 `chrome://`, 브라우저 설정, Chrome 웹스토어 등에서는 확장 프로그램이 실행되지 않습니다.

**TTS 소리가 나지 않나요?**  
시스템 볼륨과 아랍어 음성 팩을 확인하세요. 브라우저와 OS마다 지원이 다릅니다.

**아랍어를 감지하지 못합니다.**  
이미지로 된 텍스트나 비표준 인코딩은 감지되지 않을 수 있습니다.

**번역이 작동하지 않나요?**  
인터넷 연결이 필요합니다. Bing이 실패하면 설정에서 Google으로 바꿔 보세요.

**Arabic Reader로 PDF를 여는 방법은?**  
팝업의「PDF 뷰어 열기」, PDF 링크 오른쪽 클릭, 또는「PDF 스마트 감지」를 사용하세요.
