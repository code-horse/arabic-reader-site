# Arabic Reader - ഉപയോക്തൃ മാർഗ്ഗനിർദ്ദേശം

> പതിപ്പ്: v1.0.1

മലയാളം പരിഭാഷ തയ്യാറാക്കുന്നു. ഇപ്പോൾ ഇംഗ്ലീഷ് ഗൈഡിന്റെ ചുരുക്കം താഴെ നൽകിയിരിക്കുന്നു.

## ദ്രുത തുടക്കം

1. ബ്രൗസറിൽ Arabic Reader Extension ഇൻസ്റ്റാൾ ചെയ്യുക.
2. അറബിക് ഉള്ള വെബ് പേജ് തുറക്കുക.
3. Popup ൽ **Enable Tashkeel** ഓൺ ചെയ്യുക.
4. ആവശ്യമെങ്കിൽ **Whole Page Tashkeel** ഓൺ ചെയ്യുക.

## പ്രധാന സവിശേഷതകൾ

- അറബിക് വാക്കുകൾക്ക് സ്വയം Tashkeel
- Hover dictionary (അർത്ഥം, root, I'rab forms)
- Text-to-speech, karaoke highlighting
- തിരഞ്ഞെടുക്കുന്ന ടെക്സ്റ്റിന് translation
- Built-in PDF reader, PDF smart detection
