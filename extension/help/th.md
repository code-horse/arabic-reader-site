# Arabic Reader - คู่มือการใช้งาน

> เวอร์ชัน: v1.0.0

## บทนำ

Arabic Reader เป็นส่วนขยายเบราว์เซอร์ที่ออกแบบมาสำหรับผู้เรียนภาษาอาหรับ โดยเพิ่ม Tashkeel (เครื่องหมายสระ / حركات) ให้กับคำภาษาอาหรับบนหน้าเว็บและ PDF โดยให้การออกเสียงอัตโนมัติ พจนานุกรมโฮเวอร์พร้อมการวิเคราะห์รากและรูปแบบไวยากรณ์ คุณสมบัติการแปลงข้อความเป็นคำพูด และการแปล — ช่วยให้คุณเรียนรู้การออกเสียงและไวยากรณ์ภาษาอาหรับได้ง่ายขึ้น

---

## คุณสมบัติหลัก

- **อัตโนมัติ Tashkeel** — เพิ่มเครื่องหมายสระ (fathah, kasrah, dammah, sukun, shadda, tanween) ให้กับคำภาษาอาหรับบนหน้าเว็บใดๆ
- **โหมดทั้งหน้า** — คลิกเพียงครั้งเดียว Tashkeel เพื่อดูคำภาษาอาหรับทั้งหมดบนหน้า
- **พจนานุกรมโฮเวอร์** — วางเมาส์เหนือคำเพื่อดูการออกเสียง คำจำกัดความ การวิเคราะห์ราก และแบบฟอร์ม I'rab เลือกระหว่างโหมดพจนานุกรม โหมดคำแนะนำเครื่องมือ หรือปิด
- **I'rab การวิเคราะห์** — แสดงการลงท้ายด้วยตัวพิมพ์ตามหลักไวยากรณ์ (นาม, กรรมกริยา, สัมพันธการก) สำหรับคำที่เกี่ยวข้อง
- **การอ่านออกเสียงข้อความ** — TTS ภาษาอาหรับพร้อมการเน้นคาราโอเกะแบบคำต่อคำ (เสียง ar-SA)
- **การเลือกคำพูด** — เลือกข้อความใดก็ได้ แถบเครื่องมือขนาดกะทัดรัดจะปรากฏขึ้นพร้อมปุ่มพูดและแปล
- **การแปล** — เลือกข้อความใดก็ได้ คลิกแปลเพื่อรับการแปลทันทีผ่าน Bing หรือ Google Translate (108 ภาษา)
- **โปรแกรมอ่าน PDF** — โปรแกรมอ่าน PDF RTL ในตัวพร้อม Tashkeel รองรับคำอธิบายประกอบ
- **การตรวจจับอัจฉริยะ** — การตรวจจับข้อความภาษาอาหรับอัตโนมัติและการเปลี่ยนเส้นทางอัจฉริยะ PDF
- **การสกัดราก** — ระบุรากของคำเพื่อความเข้าใจที่ดีขึ้นเกี่ยวกับสัณฐานวิทยาของภาษาอาหรับ
- **แป้นพิมพ์ลัด** — เข้าถึงฟีเจอร์หลักได้อย่างรวดเร็ว
- **อินเทอร์เฟซหลายภาษา** — UI ภาษาอาหรับ อังกฤษ และจีน

## วิธีการใช้งาน

### ขั้นตอนที่ 1: ติดตั้งส่วนขยาย

ติดตั้ง **Arabic Reader** จาก [Chrome Web Store](https://chromewebstore.google.com/) หรือโหลดในเครื่องในโหมดนักพัฒนาซอฟต์แวร์

### ขั้นตอนที่ 2: เปิดหน้าเว็บใดก็ได้

เยี่ยมชมหน้าเว็บที่มีเนื้อหาภาษาอาหรับ

### ขั้นตอนที่ 3: เปิดใช้งาน Tashkeel

คลิกไอคอนส่วนขยายในแถบเครื่องมือของเบราว์เซอร์ สลับ "เปิดใช้งาน Tashkeel" จากนั้นสลับ "ทั้งหน้า Tashkeel" เพื่อใส่คำอธิบายประกอบคำภาษาอาหรับทั้งหมดบนหน้า

### ขั้นตอนที่ 4: ดูคำอธิบายประกอบ

วางเมาส์เหนือคำเพื่อดูคำแนะนำเครื่องมือ Tashkeel พร้อมเสียงพูด คำจำกัดความ และการออกเสียง คลิกไอคอนลำโพงเพื่อฟังคำนั้น

### ขั้นตอนที่ 5: พูดและแปลข้อความที่เลือก

เลือกข้อความภาษาอาหรับด้วยเมาส์ของคุณ แถบเครื่องมือขนาดเล็กจะปรากฏขึ้นใกล้กับส่วนที่เลือกโดยมีปุ่มสองปุ่ม:
- **ลำโพง** — อ่านออกเสียงข้อความที่เลือกพร้อมไฮไลต์แบบคำต่อคำแบบคาราโอเกะ
- **แปล** — แสดงกรอบคำแปลแบบอินไลน์ใต้แถบเครื่องมือ

> **เคล็ดลับ:** คลิกไอคอนส่วนขยายในแถบเครื่องมือของเบราว์เซอร์เพื่อเปิดแผงการตั้งค่าและปรับโหมดการวางเมาส์เหนือ อัตราคำพูด เครื่องมือการแปล และอื่นๆ

## ทั้งหน้า Tashkeel โหมด

เมื่อเปิดใช้งานโหมดทั้งหน้า Tashkeel ทุกคำภาษาอาหรับบนหน้าจะถูกเพิ่มเครื่องหมายสระ ส่วนขยายใช้การตรวจจับอัจฉริยะเพื่อระบุบล็อกข้อความภาษาอาหรับและใช้ Tashkeel โดยใช้พจนานุกรมที่รวมมาและอัลกอริธึม NLP

---

## พจนานุกรมโฮเวอร์

ส่วนขยายนี้ประกอบด้วยพจนานุกรมภาษาอาหรับในตัว คุณสามารถเลือกจากโหมดโฮเวอร์หลายโหมดในการตั้งค่า:

| โหมด | พฤติกรรม |
|------|----------|
| **พจนานุกรม** | โฮเวอร์แสดง Tashkeel + คำจำกัดความ + root + I'rab + ปุ่มการออกเสียง |
| **คำแนะนำเครื่องมือ** | โฮเวอร์แสดง Tashkeel + ปุ่มการออกเสียง (ไม่มีคำจำกัดความ) |
| **ปิด** | ไม่มีเอฟเฟกต์โฮเวอร์ |

ใน **โหมดพจนานุกรม** คำแนะนำเครื่องมือจะแสดง:
- คำที่มีการเปล่งเสียงเต็ม (Tashkeel)
- ปุ่มการออกเสียง (คลิกเพื่อฟัง)
- คำจำกัดความของคำ
- การวิเคราะห์ราก (รากสามตัวอักษร)
- I'rab (ตัวพิมพ์ไวยากรณ์) เมื่อเปิดใช้งาน

## โปรแกรมอ่าน PDF

Arabic Reader มีโปรแกรมอ่าน PDF ในตัวพร้อมการสนับสนุน RTL เต็มรูปแบบที่ช่วยให้คุณสามารถอ่านเอกสาร PDF ภาษาอาหรับพร้อมคำอธิบายประกอบ Tashkeel

### กำลังเปิด PDF

**วิธีที่ 1: จากป๊อปอัป**  
คลิกไอคอนส่วนขยาย จากนั้นคลิก "Open PDF Reader" ลากและวางไฟล์ PDF หรือคลิก "เลือกไฟล์" เพื่อเปิด PDF ในเครื่อง คุณยังสามารถวาง URL PDF ได้

**วิธีที่ 2: เมนูบริบท**  
คลิกขวาที่ลิงก์ `.pdf` บนหน้าเว็บแล้วเลือก "เปิด PDF ด้วย Arabic Reader"

**วิธีที่ 3: การตรวจจับอัตโนมัติ**  
เมื่อเปิดใช้งาน "PDF Smart Detection" ในการตั้งค่า ส่วนขยายจะเปลี่ยนเส้นทาง URL `.pdf` ไปยังเครื่องอ่านในตัวโดยอัตโนมัติ

### คุณสมบัติโปรแกรมอ่าน PDF

- **Tashkeel คำอธิบายประกอบ** — คุณสมบัติคำอธิบายประกอบทั้งหมดใช้งานได้กับข้อความ PDF
- **คลิกพจนานุกรม** — คลิกที่คำใดก็ได้เพื่อดูคำจำกัดความ
- **แถบเครื่องมือการเลือก** — เลือกข้อความที่จะพูด แปล หรือคัดลอก
- **แถบด้านข้าง** — โครงร่างสารบัญและภาพขนาดย่อของหน้า
- **ค้นหา** — ค้นหาข้อความในรูปแบบ PDF
- **ธีม** — ธีมการอ่านสีเข้ม สว่าง และซีเปีย
- **ซูม** — ระดับการซูมหลายระดับ รวมถึงอัตโนมัติ การปรับพอดีหน้า และความกว้างของหน้า
- **เค้าโครง RTL** — รองรับขวาไปซ้ายเต็มรูปแบบ

## ข้อความเป็นคำพูด

ส่วนขยายใช้เสียง TTS ภาษาอาหรับในตัวของ Chrome (ar-SA) สำหรับการออกเสียง

**คำเดียว:** วางเมาส์เหนือคำแล้วคลิกปุ่มลำโพงในเคล็ดลับเครื่องมือ

**คำพูดสำหรับการเลือก:** เลือกข้อความภาษาอาหรับใดก็ได้ แถบเครื่องมือแบบลอยจะปรากฏขึ้นพร้อมปุ่มลำโพง คลิกเพื่ออ่านออกเสียงพร้อมไฮไลท์คำต่อคำสไตล์คาราโอเกะ

**เมนูคลิกขวา:** เลือกข้อความ คลิกขวาและเลือก "พูดสิ่งที่เลือก"

**แป้นพิมพ์ลัด:** เลือกข้อความแล้วกด `Alt+Shift+S` (Mac: `Ctrl+Shift+S`)

## แปล

เลือกข้อความใดก็ได้บนหน้าและใช้คุณสมบัติการแปลเพื่อรับการแปลทันที

**วิธีที่ 1: แถบเครื่องมือการเลือก**  
เลือกข้อความ จากนั้นคลิกปุ่มแปลในแถบเครื่องมือ

**วิธีที่ 2: เมนูคลิกขวา**  
เลือกข้อความ คลิกขวาและเลือก "การเลือกการแปล"

**วิธีที่ 3: แป้นพิมพ์ลัด**  
เลือกข้อความแล้วกด `Alt+Shift+T` (Mac: `Ctrl+Shift+T`)

**เครื่องมือแปล:**
- **Bing Translate** (ค่าเริ่มต้น) — ขับเคลื่อนโดย Microsoft Translator
- **Google Translate** — ขับเคลื่อนโดย Google

เอ็นจิ้นทั้งสองรองรับ **108 ภาษาเป้าหมาย**

## แป้นพิมพ์ลัด

| ทางลัด | ทางลัด Mac | การกระทำ |
|----------|-------------|--------|
| `Alt+Shift+A` | `Ctrl+Shift+A` | สลับ Tashkeel เปิด/ปิดคำอธิบายประกอบ |
| `Alt+Shift+S` | `Ctrl+Shift+S` | พูดข้อความที่เลือก |
| `Alt+Shift+T` | `Ctrl+Shift+T` | แปลข้อความที่เลือก |

> **เคล็ดลับ:** คุณสามารถปรับแต่งทางลัดเหล่านี้ใน Chrome ได้ที่ `chrome://extensions/shortcuts`

## คู่มือการตั้งค่า

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **เปิดใช้งาน Tashkeel** | สวิตช์หลักเพื่อเปิดหรือปิดใช้งานคุณลักษณะคำอธิบายประกอบ Tashkeel |
| **ทั้งหน้า Tashkeel** | เมื่อเปิดใช้งาน จะเพิ่มเครื่องหมายสระให้กับคำภาษาอาหรับทั้งหมดบนหน้า |
| **โหมดโฮเวอร์** | เลือกลักษณะการทำงานของโฮเวอร์: พจนานุกรม (คำจำกัดความ + เสียง), เคล็ดลับเครื่องมือ (Tashkeel + เสียง) หรือปิด |
| **แสดง I'rab แบบฟอร์ม** | แสดงการลงท้ายด้วยตัวพิมพ์ไวยากรณ์สำหรับคำที่เกี่ยวข้อง |
| **อัตราการพูด** | ปรับความเร็วในการอ่านประโยค |
| **เครื่องมือแปล** | เลือกระหว่าง Bing Translate และ Google Translate |
| **ภาษาเป้าหมาย** | ตั้งค่าภาษาเป้าหมายการแปล |
| **PDF การตรวจจับอัจฉริยะ** | เมื่อเปิดใช้งาน จะเปลี่ยนเส้นทาง URL ของ PDF ไปยังโปรแกรมอ่านในตัว | โดยอัตโนมัติ

## คำถามที่พบบ่อย

**ถาม: ทำไมบางหน้าถึงใช้งานไม่ได้**  
ตอบ: ด้วยเหตุผลด้านความปลอดภัย ส่วนขยายเบราว์เซอร์ไม่สามารถทำงานบนหน้าพิเศษ เช่น `chrome://` การตั้งค่าเบราว์เซอร์ หรือ Chrome Web Store

**ถาม: ไม่มีเสียงจากการอ่านออกเสียงข้อความ**  
ตอบ: โปรดตรวจสอบการตั้งค่าระดับเสียงของระบบและตรวจดูให้แน่ใจว่าได้ติดตั้งชุดเสียงภาษาอาหรับแล้ว การรองรับคำพูดจะแตกต่างกันไปตามเบราว์เซอร์และระบบปฏิบัติการ

**ถาม: ส่วนขยายตรวจไม่พบข้อความภาษาอาหรับ**  
ตอบ: ส่วนขยายใช้การตรวจจับอัจฉริยะเพื่อค้นหาบล็อกข้อความภาษาอาหรับ หากข้อความแสดงผลเป็นรูปภาพหรือใช้การเข้ารหัสที่ไม่เป็นไปตามมาตรฐาน ระบบอาจตรวจไม่พบข้อความนั้น

**ถาม: การแปลไม่ทำงานใช่ไหม**  
ตอบ: การแปลต้องใช้การเชื่อมต่ออินเทอร์เน็ต หาก Bing Translate ล้มเหลว ให้ลองสลับไปที่ Google Translate ในการตั้งค่า

**ถาม: ฉันจะเปิด PDF ด้วย Arabic Reader ได้อย่างไร**  
ตอบ: คลิก "เปิด PDF Reader" ในป๊อปอัป คลิกขวาที่ลิงก์ PDF แล้วเลือก "เปิด PDF ด้วย Arabic Reader" หรือเปิดใช้งาน "PDF Smart Detection" ในการตั้งค่า
