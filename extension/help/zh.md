# Arabic Reader - 使用手册

> 版本：v1.5.0

## 简介

Arabic Reader 是一款面向阿拉伯语学习者的浏览器扩展。它为网页和 PDF 中的阿拉伯语单词添加元音标记（تشكيل），提供多语言悬停词典（包含英语/法语释义、词根分析和语法形式）、文本朗读和翻译功能，帮助你更轻松地学习阿拉伯语发音和语法。

---

## 主要功能

- **自动标注** — 为任意网页上的阿拉伯语单词添加元音标记（فتحة、كسرة、ضمة、سكون、شدة、تنوين）
- **全网页模式** — 一键为页面上所有阿拉伯语单词添加标注
- **悬停词典** — 鼠标悬停查看元音标注、英语或法语释义、词根分析和语法形式；可选词典模式、提示模式或关闭
- **语法分析** — 显示阿拉伯语的语法变化形式（主格、宾格、属格）
- **文本朗读** — 阿拉伯语 TTS，逐词卡拉 OK 高亮（ar-SA 语音）
- **选区朗读** — 选中文本后出现浮动工具栏，包含朗读和翻译按钮
- **翻译** — 选中文本，点击翻译即可通过必应或谷歌获取即时翻译（108 种语言）
- **PDF 阅读器** — 内置 RTL PDF 查看器，支持元音标注
- **智能检测** — 自动检测阿拉伯语文本和 PDF 智能重定向
- **词根提取** — 识别词根，帮助理解阿拉伯语词法
- **键盘快捷键** — 快速访问核心功能
- **多语言界面** — 支持阿拉伯语、英语和中文

## 使用方法

### 第一步：安装扩展

从 [Chrome 网上应用店](https://chromewebstore.google.com/) 安装 **Arabic Reader**，或在开发者模式下本地加载。

### 第二步：打开网页

访问任何包含阿拉伯语内容的网页。

### 第三步：启用标注

点击浏览器工具栏中的扩展图标，打开"启用标注"开关，然后打开"全网页标注"为所有阿拉伯语单词添加标注。

### 第四步：查看标注

将鼠标悬停在单词上查看元音标注、释义和发音。点击扬声器图标听发音。

### 第五步：朗读和翻译

用鼠标选中阿拉伯语文本，会出现浮动工具栏：
- **扬声器按钮** — 朗读选中文本，逐词卡拉 OK 高亮
- **翻译按钮** — 显示行内翻译气泡

## 悬停词典

扩展内置阿拉伯语词典，可在设置中选择不同悬停模式：

| 模式 | 行为 |
|------|------|
| **词典** | 悬停显示 Tashkeel + 释义 + 词根 + I'rab + 发音按钮 |
| **提示** | 悬停显示 Tashkeel + 发音按钮（无释义） |
| **关闭** | 不显示悬停效果 |

释义语言选项：

| 语言 | 数据来源 |
|------|----------|
| 英语 | Wiktionary 阿拉伯语词条的本地子集 |
| 法语 | Wiktionary 阿拉伯语词条的本地子集 |

> **提示：** 词典数据随扩展本地打包并按需加载，查词过程离线完成；只有你主动使用翻译功能时才会访问第三方翻译服务。

## 键盘快捷键

| 快捷键 | Mac 快捷键 | 操作 |
|--------|-----------|------|
| `Alt+Shift+A` | `Ctrl+Shift+A` | 开关元音标注 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 朗读选中内容 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 翻译选中内容 |

> **提示：** 可在 Chrome `chrome://extensions/shortcuts` 页面自定义快捷键。

## 设置说明

| 设置项 | 说明 |
|--------|------|
| **启用标注** | 主开关，启用或禁用元音标注功能 |
| **全网页标注** | 启用后，为页面上所有阿拉伯语单词添加标注 |
| **悬停模式** | 选择悬停行为：词典（释义+发音）/ 提示（标注+发音）/ 关闭 |
| **释义语言** | 设置词典释义语言：英语或法语 |
| **显示语法形式** | 显示适用单词的语法变化形式 |
| **朗读语速** | 调整句子朗读速度 |
| **翻译引擎** | 选择必应翻译或谷歌翻译 |
| **目标语言** | 设置翻译目标语言 |
| **PDF 智能检测** | 启用后，自动将 PDF URL 重定向到内置阅读器 |

## 常见问题

**问：为什么在某些页面上不起作用？**  
答：出于安全原因，浏览器扩展无法在 `chrome://`、浏览器设置或 Chrome 网上应用店等特殊页面上运行。

**问：没有声音？**  
答：请检查系统音量设置，确保已安装阿拉伯语语音包。TTS 支持因浏览器和操作系统而异。

**问：翻译不可用？**  
答：翻译需要互联网连接。如果必应翻译失败，请在设置中切换到谷歌翻译。

**问：如何打开 PDF？**  
答：点击弹出菜单中的"打开 PDF 阅读器"，右键 PDF 链接选择"使用 Arabic Reader 打开"，或启用"PDF 智能检测"自动重定向。

**问：词典可以离线使用吗？**

答：可以。内置 Tashkeel 词典以及基于 Wiktionary 的英语/法语释义子集都存储在扩展本地，查词不会访问网络。

**问：可以切换词典释义语言吗？**

答：可以。在弹出设置中将“悬停模式”设为“词典”，再把“释义语言”设为英语或法语。
